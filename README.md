### Installation

Using [npm](https://www.npmjs.com/):

    $ npm install


Create your .env file

```
PARSE_APPLICATION_ID=
PARSE_JAVASCRIPT_KEY=
FACEBOOK_APP_ID=
GOOGLE_ANALYTICS_ID=
```


### Development ( Run )

#### Run

    $ npm start


### Production ( Build & Run )

#### Build

    $ npm run dist

#### Run

    $ npm run server
